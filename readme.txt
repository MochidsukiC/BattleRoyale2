# Battle Royale2
Corsica バトロワ案(2)

存在する概念:マップ,破壊不可建造物,攻撃,回復,ドロップシップ,トロッコ,クラフト,収縮するワールドボーダー,ポーション,サプライボックス,

攻撃手段:TNT,ファイヤーボール,剣,弓,クロスボウ,ポーション,溶岩

マップ:
専用マップを使用。

マップには各地にギミックがある。
1.FLYパッド
上に乗ると高く舞い上がり、そこからエリトラで滑空できる。(見てればわかるので説明不要)
2.レッドストーンエレベーター
低い場所から高いところに行けるエレベーター。コマンドを一切使用しない。(見た目もろそれなので説明不要)
3.弱重力エリア
このエリア内のみ重力が弱くなる(説明あってもなくてもよい)
4.TNTキャノン砲
その名の通りTNTキャノン砲四方向に発射可能だが同時に発射出来るのは一方向だけである。ここに到達するにはアスレチックをするしかない。(説明いるかも)
5.モンスタースポナー
ブレイズのモンスタースポナーがある。
普通に沸く。
ただし、5体を超えるとランダムに1体killされる
6.金床、鍛冶台
射線通り放題の場所にある。
必要xpはガチャで出すこと、敵をkillすることで入手する
7.ガチャ
ランダムな量のXPを出すことができる。引くと爆音が出る。

サプライボックスからは、以下のものがドロップする。
エンチャント本、武器、装備各種、ポーション、回復アイテム、ごく稀にネザライトインゴット

回復アイテム:
回復アイテムには、金リンゴ(稀)、ポーション、上位金リンゴ(ごく稀)、金にんじん(稀)、ステーキ(たまに)、リンゴ(普通)、にんじん(普通)、焼いた昆布(普通)がある。各種アイテム性能に変更はない。

ポーション:
ポーションには各種レベル1までのバフポーションがドロップする。

武器:
木〜ダイアモンドまでの剣、および装備がドロップする。また、弓、クロスボウ、矢8本セットがドロップし、矢には、レベル1までのデバフ効果の矢が稀にドロップする。また、ファイアボール、溶岩バケツもドロップする
